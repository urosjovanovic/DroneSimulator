# DroneSimulator
Seminarski rad na temu "Simulacija kretanja drona" iz kursa Naucno izracunavanje, MATF Beograd, letnji semestar 2015.
# Upustva za razvoj
* Koristiti Unity verziju 5.1.1.f1 ili novije.
* Unutar  Unity-a omoguciti da meta podaci budu vidljivi i serijalizovanje u tekstualnom formatu
	* Edit -> Project Settings -> Editor
	* Version Control Mode: Visible Meta Files
	* Asset Serialization Mode: Force Text
* Koristiti Windows znak za novi red CR-LF
	* U Visual Studio 2013: File -> Advanced Save Options -> Line Endings
* Poštovati foldersku strukturu, konvenciju zvanja promenljivih i datoteka
* Pisati summary za složenije metode
* Komitovati iskljucivo stvari neophodne za kompilaciju
